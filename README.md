Day OS 🗓️
A personal productivity operating system built as a single HTML file. No frameworks, no installs, no accounts — just open and go.
Features
	•	Today View — live schedule banner, habit progress, and prioritized task list at a glance
	•	Task Manager — high / medium / low priority system with time estimates and due dates
	•	Habit Tracker — daily streaks, weekly view, and one-tap check-ins
	•	Schedule — time-blocked daily schedule with past / present / upcoming indicators
	•	Focus Timer — stopwatch with task tagging and session logging
Stack
	•	Vanilla HTML, CSS, JavaScript
	•	Zero dependencies
	•	Mobile-first, PWA-ready (add to iOS home screen via Safari)
	•	Data persists via localStorage
Usage
Open DayOS.html in any browser, or visit the live version:
🔗 scrigot.github.io/daios/DayOS.html
To add to your iPhone home screen:
	1	Open the link in Safari
	2	Tap the Share button → Add to Home Screen
	3	Tap Add
